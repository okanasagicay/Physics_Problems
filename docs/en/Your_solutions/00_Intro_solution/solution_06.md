TASK 06 – FUNCTION ANALYSIS

Problem:
f(x) = 3x² - 12x + 7
Find local maxima or minima.

Solution:

First derivative:
f'(x) = 6x - 12

Set f'(x) = 0:
6x - 12 = 0
x = 2

Second derivative:
f''(x) = 6

Since f''(2) = 6 > 0, x = 2 is a local minimum.

Minimum value:
f(2) = 3(2)² - 12(2) + 7
     = 3(4) - 24 + 7
     = 12 - 24 + 7
     = -5

Final answer:
Local minimum at (2, -5)
No local maximum
