TASK 01 – VECTOR ALGEBRA

Problem:
Given vectors a = [2, 1, -3] and b = [4, -2, 1], calculate:
a) Magnitude of each vector
b) Dot product a · b
c) Cross product a × b
d) Angle between vectors

Solution:

a) Magnitudes:
|a| = √(2² + 1² + (-3)²) = √(4 + 1 + 9) = √14
|b| = √(4² + (-2)² + 1²) = √(16 + 4 + 1) = √21

b) Dot product:
a · b = (2)(4) + (1)(-2) + (-3)(1) = 8 - 2 - 3 = 3

c) Cross product:
a × b = [ (1)(1) - (-3)(-2), (-3)(4) - (2)(1), (2)(-2) - (1)(4) ]
       = [ 1 - 6, -12 - 2, -4 - 4 ]
       = [ -5, -14, -8 ]

d) Angle:
cos θ = (a · b) / (|a|·|b|) = 3 / (√14 · √21) = 3 / √294 = 3 / (7√6)
θ = arccos(3 / (7√6))

Final answers:
|a| = √14, |b| = √21
a · b = 3
a × b = [-5, -14, -8]
θ = arccos(3/(7√6))
